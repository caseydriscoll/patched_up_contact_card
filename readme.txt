A widget that produces any amount of sidebar links to your social profiles

== Description ==

Patched Up Contact Card is a clean and semantic way of displaying as many of your social media sites as you are comfortable with.

= Supported Platforms =
*Facebook
*Twitter
*Skype
*Linkedin
*Reddit
*Github

*Email
*Mobile Phone
*Desk Phone
*Computer

== Installation ==

1. Upload the 'patched_up_contact_card/' folder to the /wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag widget in twidget section to finish install
4. Customize social buttons by selecting platforms and entering urls
